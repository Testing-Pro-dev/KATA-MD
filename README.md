# 🔥 KATA-MD WhatsApp Bot

> The Dripped Out WhatsApp Bot with Web Pairing

## Features
- 📥 Download YouTube, TikTok, Instagram, Facebook videos
- 🎵 Download songs/audio
- 👥 Group tools (tagall, activeusers)
- 🔐 Owner controls (ban, unban, private mode)
- 🌐 Web pairing — no QR code needed
- ⚡ Fast & lightweight

## Deploy to Railway (Free)

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template)

1. Fork this repo
2. Go to railway.app → New Project → Deploy from GitHub
3. Add environment variable: `OWNER_NUMBER=27743266789`
4. Deploy!
5. Visit your Railway URL to pair your WhatsApp

## Local Setup

```bash
npm install
node server.js
```

Visit `http://localhost:3000` to pair.

## Commands

| Command | Description |
|---------|-------------|
| `.menu` | Show all commands |
| `.yt (link)` | Download YouTube video |
| `.fb (link)` | Download Facebook video |
| `.insta (link)` | Download Instagram reel |
| `.song (query)` | Download song |
| `.yt-search (query) mp3/mp4` | Search & download |
| `.ping` | Bot speed |
| `.runtime` | Bot uptime |
| `.tagall` | Tag all group members |
| `.activeusers` | Most active users |
| `.ban (number)` | Ban user (owner only) |
| `.unban (number)` | Unban user (owner only) |
| `.private` | Private mode (owner only) |
| `.public` | Public mode (owner only) |
| `.setpfp (url)` | Set bot profile pic |
| `.setprefix (symbol)` | Change prefix |
